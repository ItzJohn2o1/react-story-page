# The Three Little Pigs - React Story Page

A React application displaying the classic fairy tale "The Three Little Pigs" with component-based architecture.

## Description

This project is built using React and demonstrates component-based design patterns. The story is split into three main components:
- **Header**: Displays the story title
- **Content**: Contains the story narrative (2-3 paragraphs)
- **Footer**: Shows the moral of the story
